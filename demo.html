Keyboard Shortcuts

  • ? - Show keyboard shortcuts
  • 0 - Show all slots
  • 1...9 - Load group
  • d - Start demo mode
  • e - Open group editor
  • t - Toggle TV view
  • i - Change tv display mode
  • f - Toggle fullscreen
  • g - Toggle image backgrounds
  • v - Toggle video backgrounds
  • n - Extended Name editor
  • s - Device configuration editor
  • q - Show QR code
  • esc - reload micboard
